Console Web
===========

Production web console for operators, supervisors, and analysts.
The current static UI at repo root is a visual prototype; this app would replace it.

Scope
-----
- Map workspace, ticketing, and telemetry panels
- Operator actions, workflows, and approvals
- Role-based access and audit overlays
